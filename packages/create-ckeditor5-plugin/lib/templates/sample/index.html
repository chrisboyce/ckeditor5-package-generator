CKEditor 5 – Development Sample

  • DLL Sample

This is a demo of the classic editor build that contains MyPlugin, which is an example plugin. You can use this sample to validate whether your custom build works fine. To use the example plugin and insert text into the editor, click on the CKEditor5 icon that is located leftmost on the toolbar.

Correct plugin file structure:
  • .js files in the src/ directory
  • .css files in the theme/ directory
  • .svg files in the theme/icons/ directory 
ckeditor5-plugin/
├─ ...
├─ src/
│  ├─ index.js
│  ├─ ....js
├─ ...
├─ theme/
│  ├─ icons/
│  │  ├─ icon.svg
│  │  ├─ ....svg
│  ├─ style.css
│  ├─ ....css
├─ ...
